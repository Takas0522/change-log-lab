# 高機能ToDoアプリケーションの作成

下記の仕様を満たすアプリケーションを作成します
- タスクの追加、削除、完了状態の切り替えができる
- タスクはタイトルと説明を持つ
- タスクは期限を設定できる
- タスクは優先度を設定できる
- タスクはタグを設定できる
- タスクは検索機能を持つ
- タスクはフィルタリング機能を持つ
- タスクはソート機能を持つ
- タスクはダッシュボード表示ができる
- タスクは通知機能を持つ
- タスクは進捗状況表示機能を持つ

# システム仕様

- フロントエンド: React
- バックエンド: ASP.NET Core WebAPI
- データベース: PostgreSQL
- 認証: JWT
