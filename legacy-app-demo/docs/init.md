# ToDoアプリケーションの構築

このアプリケーションでは下記の機能が実装されます。

- タスクの追加
- タスクの完了
- タスクの削除
- タスクの編集
- タスクのフィルタリング
- タスクの保存
- タスクの読み込み
- タスクの並び替え
- タスクの検索
- タスクの優先順位設定
- タスクの期限設定
- タスクの通知設定
- タスクのコメント機能

# 開発環境

レガシーなアプリケーションとして構成されるため
ASP.NET WebFormsを使用して構築されます（.NET Framework 4.8）
開発環境はVisual Studio 2019を使用します。
データベースはSQL Serverを使用します。
