# 発注管理クライアントアプリ

卸売・小売業向けの発注管理 Windows デスクトップアプリケーションです。
1〜5名程度の小規模チームを対象とし、スタンドアロン（1台で完結）で動作します。
商品マスタ・仕入先管理から発注の作成・承認・入荷確認まで、発注業務に必要な一連のフローをカバーします。

---

## 使用環境

| 項目 | 内容 |
|------|------|
| フレームワーク | .NET 10 / WPF Application |
| データベース | SQLite（ローカル、オフライン動作） |
| 対応 OS | Windows 10 以降 |
| 配布方法 | インストーラー（手動配布・手動更新） |
| 言語 | 日本語のみ |

---

## ユーザー・認証

- **ログイン認証** — シンプルなパスワード認証
- **ユーザーロール** — 3 種類
  - `一般ユーザー` — 発注の作成・編集・参照
  - `承認者` — 金額基準を超えた発注の承認・却下
  - `管理者` — 全操作 ＋ マスタ管理・ユーザー管理・設定変更
- **起動時画面** — ダッシュボード

## 機能一覧

### ダッシュボード

- 未処理発注件数・今月の発注金額・在庫アラート件数などのサマリー表示
- 発注状況の棒グラフ・折れ線グラフ（月別・商品別）
- アプリ起動時に在庫発注点アラートをダイアログで通知

### 発注管理

- **発注一覧** — ステータス・日付範囲での絞り込み・ページング（10 / 50 / 100 件）
- **発注作成・編集・削除** — 1 件の発注に複数商品の明細行を追加可能
- **一括発注** — 複数商品をまとめて 1 画面で発注
- **発注テンプレート** — よく使う発注パターンを保存して再利用
- **発注番号** — 自動連番（例: PO-0001）
- **発注ステータス** — 未処理 / 処理中 / 入荷待ち / 部分入荷 / 完了 / キャンセル
- **金額計算** — 単価 × 数量を自動計算（税抜管理・税込併記）
- **データ削除** — 論理削除（削除フラグ管理・データは無期限保持）
- **キーボードショートカット** — Ctrl+N（新規）/ Ctrl+S（保存）など基本操作

### 発注承認ワークフロー

- 金額が設定した閾値を超えた発注には承認者の承認が必要（1 段階承認）
- ステータス遷移: 申請中 → 承認済み / 却下
- 承認依頼・承認完了時に Windows トースト通知を送信

### 商品マスタ管理

- 商品コード・商品名・単価・単位・備考の管理
- 1 階層のカテゴリ分類
- 発注点（在庫閾値）の設定
- CSV インポートによる一括登録

### 仕入先管理

- 仕入先マスタ（会社名・連絡先の基本情報）
- 商品ごとに優先仕入先を 1 社登録
- 仕入先ごとの商品単価管理（同一商品で仕入先によって単価が異なる場合に対応）

### 在庫管理

- 簡易在庫管理（入荷確認時に在庫数を自動加算）
- 発注点を下回った商品を起動時にアラートダイアログで通知
- Windows トースト通知でも発注点アラートを表示

### 入荷管理

- 入荷予定日の登録
- 入荷確認（発注ステータスを「完了」に更新）

### 予算管理

- 月次・年次の発注予算上限を設定
- 予算超過時にアラートを表示

### 会計・請求管理

- 納品書・請求書の情報を発注に紐付けて入力・管理
- 金額は税抜で管理し、税込金額を常時併記（日本円のみ）

### 分析・レポート

- 月別・商品別の発注件数・発注金額の集計グラフ（棒グラフ / 折れ線グラフ）

### セキュリティ・運用

- **操作ログ** — 発注の作成・更新・削除などの主要操作を記録
- **手動バックアップ** — 任意のタイミングで SQLite データファイルをバックアップ
- **テーマ** — ライト / ダーク切り替え対応
- **会社情報設定** — 自社の会社名・住所を設定画面で管理

---

## 発注ステータス遷移

```
未処理 ──→ 処理中 ──→ 入荷待ち ──→ 部分入荷 ──→ 完了
                │                              ↑
                └──────────────────────────────┘
各ステータスから → キャンセル（論理削除）

※ 承認ワークフローが有効な場合:
未処理 → 申請中 → 承認済み → 処理中 → ...
                └→ 却下 → 未処理（差し戻し）
```

## 画面構成

| 画面 | 概要 |
|------|------|
| ダッシュボード | 起動時のホーム画面。サマリー・グラフ・アラート表示 |
| 発注一覧 | 絞り込み・ページング付きの発注リスト |
| 発注詳細 / 編集 | 明細行付きの発注フォーム |
| 商品マスタ | 商品の一覧・追加・編集・削除・CSV インポート |
| 仕入先マスタ | 仕入先の一覧・追加・編集・削除 |
| 承認画面 | 承認待ち発注の一覧と承認 / 却下操作 |
| 予算設定 | 月次・年次の予算上限設定 |
| 操作ログ | 主要操作の履歴参照 |
| 設定 | 会社情報・承認閾値・テーマ・バックアップ・ユーザー管理 |
